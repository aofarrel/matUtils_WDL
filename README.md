# matUtils WDL library
Collection of WDL workflows and tasks for [matUtils](https://usher-wiki.readthedocs.io/en/latest/matUtils.html) and matOptimize, which are part of [UShER](https://github.com/yatisht/usher). These are relatively basic implementations and do not cover all possible arguments.

This repo was split off from [Tree Nine](https://github.com/aofarrel/tree_nine) in April 2026; if you are looking for a complete phylogenetics workflow and/or pathogen clustering, you are probably better served by that repo. This one is more of a standard library.
